# Debian Setup

## Debian Minimal Installation

Bla.

## Openbox Window Manager

### Windown Manager

```bash
apt install openbox
```

### Debian Menu

```bash
apt install menu
```

### Preferences Manager for Openbox

```bash
apt install obconf
```

### Openbox Menu

```bash
apt install openbox-menu
```

### Display Manager

```bash
apt install lightdm
systemctl enable lightdm
```

### File Manager

```bash
apt install thunar
```

Or `pcmanfm`.

### Lightweight Taskbar

```bash
apt install tint2
```

### Terminal

```bash
apt install terminator
```

### Wallpaper Browser and Changing Utility for X

```bash
apt install nitrogen
```

Or `feh`.
